ОБЩАЯ ИНФОРМАЦИЯ
 

Приложение не является самодостаточным программным продуктом, а является серверным компонентом в цепи клиент-сервер, где его основной целью является обработка запросов на отправку SMS-сообщений SMS-клиентами (далее -дополнительное ПО).

SMS-клиентом является дополнительное ПО, которое поставляется по открытым ссылкам по email-у через панель «Загрузки» Приложения.

ЦЕЛИ ОБРАБОТКИ ДАННЫХ
 

Приложение запрашивает разрешение на отправку и просмотр SMS-сообщений (разрешение SEND_SMS), чтобы Приложение, по средствам дополнительное ПО могло осуществлять управление Приложением, как сервером SMS-рассылок. Тем самым, появляется возможность отправлять SMS из обычного персонального компьютера по необходимости пользователя.

Через панель «Загрузки» Приложение просит указать email пользователя для отправки дополнительного ПО, через которое он сможет управлять Приложением. При этом email пользователя Приложения никуда не записывается и не передается, а используется лишь для однократной доставки дополнительного ПО для Приложения.  

Приложение считывает локальный IP-адрес устройства, чтобы вывести его в Приложение для более удобной настройки с дополнительным ПО. Для того, чтобы вывести IP адрес, Приложение запрашивает разрешение на осуществление телефонных звонков и управление ими (разрешение READ_PHONE_STATE). Однако, Приложение использует это разрешение лишь для запроса локального IP-адреса устройства, так как без него получить доступ к данной функции невозможно.

БЕЗОАСНОСТЬ ДАННЫХ ПОЛЬЗОВАТЕЛЕЙ

 

Обработка данных пользователей происходит в Приложении на их собственных устройствах без участия разработчика.

Безопасность данных обеспечивается стандартными средствами операционной системы, в самом Приложении дополнительные средства защиты отсутствуют, кроме тех, которые заявлены в функционале Приложения, а именно:

  1. Имеет встроенный механизм авторизации по сети через логин и пароль.
  2. Механизм защиты от подбора логина и пароля с автоматической блокировкой IP адреса злоумышленника.
  3. Возможность просмотреть черный список IP адресов и снять с блокировки. Выставить время блокировки.
  4. Может ограничить суточный проход SMS-сообщений через сервер.
  5. Можно изменять логин или пароль на сервере.
  6. Можно изменять порт подключения к SMS-серверу.
