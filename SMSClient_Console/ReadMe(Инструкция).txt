 Внимание! Если в командной строке русские буквы отображаются некорреткно, то нажмите на значек командной строки в верхнем левом углу. Появится контектсное меню,
 в котором выберите пункт "свойтсва" и в них выставит шрифт "Lucida Console". Также, можете увеличить размер для шрифта.
 
 Также, на вашем ПК должен быть установлен JDK1.8 (Java), в противном случае программа не запустится.  
 
 
1. Файл start_client.bat - это консольное приложение, которое позволяет соединяться с Android SMS сервером и рассылать SMS сообзения. Чтобы его запустить вам необходимо дважды
 кликнуть по файлу "start_client.bat". Откроется командная строка, в которой вам будет предложено запустить сервис передачи SMS. 
Внимание! SMS сервер уже должен быть установлен на вашем смартфоне и подключен к локальной сети по Wi-Fi или любым другим способом.

2. Файл sms_cfg.txt - это обычный текстовый файл, который содержит настройки для подключения к SMS серверу. Вам лишь нужно выставить те же, что и на вашем SMS сервере.
Файл легко редактируется и открывается любым текстовым редактором. Он содержит в себе 4-е пункта:
ip=192.168.1.100   //IP-адрес сервера (DHCP или статический IP - присваивается вашей сетью)
port=11330         //Номер порта, котрый слушает сервер (устанавливается в настройках сервера)
login=user         //Логин для входа на сервер (устанавливается в настройках сервера)
pass=123           //Пароль для входа на сервер (устанавливается в настройках сервера)

3. Порядок работы с сервером в консоли такой:

3.1 Соглашаетесь или не соглашаетесь с запуском SMS сервиса.
3.2 Если согласились, вводите номера телефонов через точку с запятой.
3.3 Далее вводите текстовое сообщение, которое будет отправленно вышеуказанным обонентам.
3.4 Все, если сделали все правильно, запос об отправки SMS уйдет на сервер. Ну, или читайте инструкцию выше по внимательней!
